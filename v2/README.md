# Axial v2

Mobile-first narrative intelligence presented as browsable investigation packs.

## Product Surfaces

- Home discovery shelves with generated editorial pack covers
- Narrative pack detail with brief, capital, history, and evidence views
- Draggable pack relationship graph with explicit evidence posture legend
- Searchable 1,200-entity sourced ontology index
- Saved-pack library
- Two-floor floating assistant and primary navigation
- Deep-linkable pack, detail, and graph states
- Installable PWA shell

## Evidence Boundary

Documented relationships, observed narrative behavior, and modeled scenarios are visually distinct. Capital benefit scores describe outcome sensitivity and do not assert promotion, coordination, or realized profit without direct evidence.

## Run

```powershell
npm run dev
npm run build
```
